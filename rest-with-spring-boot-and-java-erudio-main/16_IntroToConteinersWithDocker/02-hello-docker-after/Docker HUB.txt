Pegando outros containers

CLIENTE | DOCKER HOST | REGISTRY(DOCKE HUB)
dockcer pull, run, build | Docker Deamon verifica se tem no docker host o container ou a simagens necessarias | se nao tiver ele vai no docker hub e pega a imagem

Enviando docker image
- criando a tag par ficar no formato do docker hub

docker tag <nome da imagem no docker host>:<versao> <nome final> ex cauzy/<nome da imagem no docker host>:<versao>

LOGIN
docker login docker.io

mandando para docker hub
docker push <img>

rodando container
docker -p <porta>:<porta> -d run <nome da imagem e versao>
- d nao excuta o console do conteiner, nao ficando preso

parando o container
docker container stop <nome do container>

mostrar os logs no container
docker logs -f <id do container>
entra dentro do container

docker pull <nome da imagem>
- baixar a img
